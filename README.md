# 关于LCLang

**LCLang**, 全称The LiChen Programming Language, 以下简称LCL.是一个~~专为李辰开发的~~, 基于cpp的傻瓜语言. 目的是把写好的LCL用开发的编译器编译成cpp语言, 以轻松解决cpp问题.

LCL编译器开发组现有两名成员, 开发进度为刚开始写功能.
